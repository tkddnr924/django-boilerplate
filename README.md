# django-boilerplate

### Virtualenv 사용

``````
virtualenv "env_name"
source env_name/bin/active
``````

### Virtualenv 해제

```
(env_name)$ deactivate
```



### PIP Requirement

파이썬 패키지 설치

```
pip install -r requirement.txt
```
